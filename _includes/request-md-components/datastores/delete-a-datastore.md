## Delete a datastore

DELETE
{:.label .label-DELETE}

/api/datastores/{datastore-id}
{:.path .path-DELETE}

Deletes the datastore `{datastore-id}`.

### Prerequisites
- Datastore `{datastore-id}` must exist

### Response
{: .d-inline-block }

204
{:.label .label-DELETE}

No response body provided.
