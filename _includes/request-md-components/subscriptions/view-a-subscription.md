## View a subscription

GET
{:.label .label-GET}

/api/subscriptions/{subscription-id}
{:.path .path-GET}

Returns the subscription `{subscription-id}`.

### Prerequisites
- Subscription `{subscription-id}` must exist

### Response
{: .d-inline-block }

200
{:.label .label-GET}

[Subscription](#subscription) object with HAL links.
